20.3 Où obtenir la dernière version de ce document

Vous pouvez trouver la dernière version de ce document sur le tri des pages de manuel Sylpheed officiel sur http://nlpagan.net/sylman/index.html.

À cet endroit, le manuel est chargé comme pages HTML. À partir de cette page, vous pouvez télécharger ces pages comme un fichier tar.gz pour la navigation hors ligne sur votre propre ordinateur.

Vous pouvez aussi piocher dans le dépôt CVS sur http://sourceforge.net et chercher le projet « sylpheeddoc ». Le document y est stocké comme un fichier SGML, vous aurez besoin des outils SGML pour celui-ci !
