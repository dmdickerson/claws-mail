4. Installation.

4.1 Environnement nécessaire

Pour réussir la compilation de Sylpheed, vous avez au moins besoin de :

  • Tout Unix compatible POSIX UNIX ou des OS similaires i.e. Linux, FreeBSD, Solaris
  • Un compilateur ANSI C récent (gcc 2.7.2.3 devrait aussi fonctionner). Note : le compilateur C de Sun ne compilera pas Sylpheed.
Optionnellement, vous avez besoin de :
  • Imlib ou gdk-pixbuf (pour la visualisation des image intégrée)
  • libcompface (pour X-Face)
  • libjconv (pour I18N)
  • GPGME (pour GnuPG)
  • OpenLDAP (pour une connection avec un serveur LDAP)
  • Jpilot (pour utiliser un carnet d'adresses Palm)
  • l'équivalent de GTK+-devel et Xfree86-devel. (Sinon ./configure échouera.)
Note: Sylpheed peut être instable en fonction de l'environnement. Veuillez utiliser les versions les plus récentes des bibliothèques dans la mesure du possible.

4.3 Éléments et points clés de l'installation

  • Téléchargez la dernière version de SYlpheed a partir de: http://sylpheed.good-day.net/
  • Décompressez les fichiers sources dans un nouveau répertoire
  • tar -xvfz sylpheed-.tar.gz (pour tar.gz)
  • tar -xvfI sylpheed-.tar.bz2 (pour tar.bz2)
  • cd vers le nouveau répertoire sylpheed
  • Lancez ./configure pour contrôler votre environnement et créer les Makefiles d'installation
  • Lancez le programme 'make' qui compilera les sources. Ne passez pas à cette étape s'il existe des erreurs fatales dans la précédente. FLes erreurs fatales indiquent que des éléments manquent de manière à compiler Sylpheed correctement.

Si le processus make est terminé, lancez "make install" (en tant que root), pour installer le programme et les fichiers de support

4.4 Installation sur Linux en utilisant RPM

Comme root, lancez la commande "rpm -ivh " sur le fichier RPM que vous avez téléchargé. Dès que ce procéssus est effectué, vous pouvez lancer Sylpheed comme utilisateur.

4.5 Installation sur Debian Linux

Avec une connexion internet :

apt-get install sylpheed (pour l'installer avec une connexion internet)

Sans connexion internet, récupérez le paquet debian, par exemple sur http://packages.debian.org/unstable/mail/sylpheed.html

dpkg -i sylpheedx.x.x-1i386.deb

dpkg --install sylpheedx.x.x-1i386.deb

4.6 Installation sur FreeBSD

utilisant l'outil /stand/sysinstall (c'est une interface basée sur les menus) :

lancez "/stand/sysinstall" en tant que root

dans le menu principal, choisissez "Configure" -> "Packages" -> "mail", puis "sylpheed" puis retournez vers la liste des "Packages", choisissez "Install"

Utiliser les portages

Allez chercher le port de sylpheed sur :

ftp://ftp.freebsd.org/pub/FreeBSD/ports/i386/packages-4.3-release/mail

puis, en tant que root : pkg-add sylpheed-0.x.x.tgz

en utilisant les ports avec compilation :

mettez d'abord votre portage à jour, traitez-le sur http://www.freebsd.org/ports/mail.html

cd /usr/ports/mail/sylpheed

make

Si cela réussit, saisissez "make install"

dans le cas où cela échoue, demandez le paquet requis, si nécessaire, (sur la même page), mettez le fichier du paquet dans /usr/ports/distfiles/ puis :

  • cd /usr/ports/mail/sylpheed
  • make
  • make install

Vous avez besoin d'une version plus à jour ?

Utilisez simplement la méthode standard d'Unix (veuillez voir "Éléments et points clés de l'installation")
