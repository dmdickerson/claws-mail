7. Vues et dialogues

7.1 Vue des dossiers

La vue des dossiers vous montre une arborescence de tous les dossiers de courriers que vous avez créés. Cliquez une fois sur le nom d'un dossier dans la vue du sommaire pour voir les messages qui y sont stockés.

Lorsque vous voyez le signe + devant le nom du dossier, cela signifie que la branche de cette partie de l'arbre est cachée. C'est-à-dire que ce dossier renferme d'autres dossiers. Vous pouvez déployer cette branche en cliquant sur le signe + pour pouvoir voir la liste des dossiers qu'elle contient. De la même manière, vous pouvez aussi refermer la branche en cliquant sur le signe - devant un dossier.

Dossiers spéciaux

Brouillon : ici, vous pouvez stocker les messages que vous voulez envoyer plus souvent ou sur lesquels vous voulez encore travailler. En utilisant la fonction Rééditer dans le menu Message, vous pouvez de nouveau éditer le message.

Corbeille : c'est ici que vont tous les messages supprimés. En fonction de la configuration des préférences générales, la corbeille est ou n'est pas vidée lorsque vous quittez Sylpheed.

7.2 Vue du sommaire

La fenêtre du sommaire montre un certain nombre d'informations. Vous pouvez choisir celles que vous voulez voir dans le menu Sommaire -> Choix des éléments affichés.... Voici une liste de toutes les options disponibles :

  • Marque : montre si le message est marqué ;
  • Non lu : montre si le message est lu ou pas ;
  • MIME : montre si le message contient des pièces jointes en MIME ;
  • Numéro : montre le numéro MH du message ;
  • Taille : montre la taille du message complet incluant les pièces jointes ;
  • Date : montre la date d'envoi par l'expéditeur ;
  • De : montre l'expéditeur qui a envoyé le message ;
  • Sujet : montre le sujet du message.
Le sommaire peut être trié en cliquant sur le bouton contenant le titre de la colonne. Cliquez une première fois pour trier dans un sens (a-z), cliquez une seconde fois pour l'autre sens (z-a).

Si la taille de la colonne n'est pas suffisante, placez le curseur de la souris sur le bord droit du titre de la colonne que vous voulez ajuster, pressez ensuite sur le bouton 1 (gauche) de la souris et tirez la colonne jusqu'à ce qu'elle ait la taille désirée.

Pour naviguer entre les messages, sélectionnez-en un en cliquant dessus avec la souris. Cela rendra en outre la fenêtre du sommaire active et vous pouvez alors vous déplacer vers le haut ou vers le bas dans la liste avec les touches de curseur ou en pressant « p » (pour previous, précédent) et « n » (pour next, suivant) comme dans Pine.

7.3 Vue du message

Dans la vue du message, vous pouvez lire le message que vous avez sélectionné dans la fenêtre du sommaire. Si vous entrez dans la fenêtre du sommaire sans sélectionner de message, la vue de message sera vide. Au dessus de celle-ci quelques informations sont affichées concernant le courrier que vous êtes en train de lire ainsi que quelques statistiques sur le dossier sélectionné.

Vous pouvez dérouler le contenu message vers le haut ou vers le bas en utilisant les touches fléchées après avoir cliqué à l'intérieur du message ou en utilisant la barre d'ascenseur.

7.4 Préférences générales

La fenêtre des préférences générales obtenue à partir du menu Configuration -> Préférences générales... dispose des onglets suivants :

Onglet de Réception

Ici, vous pouvez configurer comment recevoir les messages.

  • Vous pouvez y définir un Programme externe pour récupérer les messages à partir de la boîte aux lettres Unix par défaut. Habituellement, ce n'est pas nécessaire mais vous pouvez l'utiliser. (En fait, cette information peut être inexacte. Si quelqu'un en sait plus, qu'il me le fasse savoir.)
  • Le Spool local vous laisse définir le fichier qui contient votre boîte aux lettres locale. Vous pouvez aussi dire à Sylpheed de filtrer ou non les messages provenant de ce fichier.
  • Relève automatique du courrier : lorsque vous validez cette option, Sylpheed contrôlera régulièrement s'il y a de nouveaux courriers. Vous devez alors spécifier la périodicité.
  • Relever le courrier au démarrage signifie que Sylpheed cherchera immédiatement les nouveaux courriers sur les comptes sélectionnés lorsqu'il est chargé.
  • Dans la section Article de groupes de discussion, vous pouvez spécifier le nombre maximal d'articles qui peuvent être téléchargés lors de l'ouverture d'un groupe de discussion.

7.5 Configurer un compte/modifier un compte

Pour créer un nouveau compte dans Sylpheed, sélectionnez le menu Configuration -> Création d'un nouveau compte.... Si vous voulez modifier les options d'un compte existant, sélectionnez plutôt l'option Édition des comptes... dans le menu Configuration. Ensuite, sélectionnez le compte que vous voulez changer et enfin choisissez Éditer pour que la fenêtre de configuration du compte apparaîsse. Celle-ci propose des options réparties dans les sections suivantes :

Section Générale

Dans l'onglet de configuration Générale, vous pouvez donner un nom au compte. Ceci est pour rendre l'identification du compte plus facile.

Choisissez un nom représentatif du compte, comme le nom du fournisseur, la boîte aux lettres ou les serveurs de groupes de discussion que vous allez utiliser. Vous pouvez définir ce compte comme celui qui sera utilisé par défaut en cochant l'option Définir comme compte par défaut.

Entrez ensuite votre nom et votre adresse email et, si vous en avez une, l'organisation dans laquelle vous utilisez Sylpheed. Dans la boîte de protocole, sélectionnez le type de service que ce compte utilisera. Vous pouvez choisir parmi : POP3 (normal), POP3 (APOP auth), IMAP4, News (NNTP) ou Aucun (local). Dans le champ inbox, vous pouvez entrer le répertoire de courrier où le courrier pour ce compte sera stocké. Cette option est désactivée lorsque vous configurez un compte pour groupes de discussions (news NNTP).

Pour les services POP3 et IMAP, vous pouvez spécifier le serveur dans lequel votre courrier est stocké. Les détails sur ce nom et aussi le nom qui va dans le champ de Serveur SMPT (envoi) doivent vous être fournis par votre fournisseur d'accès.

Pour une boîte aux lettres Unix locale, le Serveur pour la réception est désactivé, car c'est votre machine qui reçoit directement les messages.

Les champs pour le Nom d'utilisateur et le Mot de passe seront aussi désactivés pour les comptes qui n'utilisent pas les services POP3 ou IMAP. Pour ces derniers, vous pouvez entrer le nom d'utilisateur et le mot de passe pour le serveur de courrier de votre fournisseur. Habituellement, ce sont les mêmes que votre nom de connexion et votre mot de passe pour accéder aux services de connexion mais cela dépend du fournisseur d'accès. En cas de doute, contactez le service support de clientèle de votre fournisseur.

Section de Réception

L'onglet Réception n'est utilisé que pour les comptes POP3. Ici, vous pouvez spécifier si les courriers qui ont été récupérés avec succès doivent être supprimés du serveur. L'option Réception de tous les messages du serveur vous permet d'obliger Sylpheed de relever non seulement les nouveaux messages mais aussi tous ceux qui ont déjà été lus (récupérés mais pas supprimés) et qui se trouvent encore sur le serveur. L'option 'Tout relever' relève le courrier pour ce compte ajoute ce compte à la liste des comptes qui sont vérifiés lorsque le bouton Tout relever de la barre d'outils est activé ou si le menu Message -> Relever le courrier de tous les comptes est sélectionné. Vous pouvez aussi spécifier ici si les messages récupérés doivent être immédiatement filtrés lors de la réception.

Section Envoyer

Dans l'onglet Envoi, vous pouvez configurer quelques options supplémentaires pour les messages que vous envoyez.

Section Composition

Dans l'onglet Composition, vous pouvez définir votre fichier de signature, entre autres.

Section Avancée

Dans l'onglet de la configuration Avancée, vous pouvez éditer les numéros des ports des serveurs que vous allez utiliser. Ceci n'est pas nécessaire, en général.
