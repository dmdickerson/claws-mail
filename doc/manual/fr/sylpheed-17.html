17. Reference

Dans cette section, vous allez trouvez quelques informations sur ce qui peut être utile pour vous rendre Sylpheed plus clair pour ce qui concerne les raccourcis claviers et une structure du format du carnet d'adresse.

17.1 Liste des raccourcis clavier

Raccourcis à une touche :

  • a : crée un nouveau message ;
  • d : efface le message sélectionné ;
  • f : fait suivre le message sélectionné ;
  • g : va dans un répertoire spécifique par le biais d'un dialogue de navigation ;
  • n : saute au message suivant/à la partie suivante d'un message multipartie ;
  • p : saute au message précédent/à la partie précédente d'un message multipartie ;
  • v : commute la vue de message on/off ;
  • w : même que un (((((((((((((()))))))))))))))
  • y : Dialogue Ouvre sauvegarde comme...

Raccourcis au menu Fichier :

  • Alt-F : ouvre le menu FICHIER ;
  • Alt-P : imprimer ;
  • Alt-W : fermer ;
  • Alt-Q : quitter.

raccourcis au menu Editer

  • Alt-E : ouvre le menu EDITER ;
  • Ctrl-C : copier ;
  • Ctrl-A : sélectionner tout ;
  • Ctrl-S : checher.

Raccourcis au menu Vue

  • Alt-V : ouvre le menu VUE.

Raccourcis au menu Message

  • Alt-M : ouvre le menu MESSAGE ;
  • Alt-I : contrôle le courrier pour tous les comptes courants ;
  • Shift-Alt-I : Contrôle les courriers pour tous les comptes ;
  • Alt-N : compose des nouveaux messages ;
  • Alt-R : répondre au message ;
  • Ctrl-Alt-R : répondre à l'émetteur (contrairement à répondre à i.e. une liste de diffusion) ;
  • Shift-Alt-R : répondre à tous (inclut toutes les adresses dans À : et Cc :) ;
  • Ctrl-F : faire suivre le message ;
  • Shift-Ctrl-F : faire suivre le message en attachement ;
  • Alt-O : déplacer le message ;
  • Alt-D : effacer le message ;
  • Shift-Ctrl-N : ouvrir le message dans une nouvelle fenêtre ;
  • Ctrl-U : voir le source du message ;
  • Ctrl-H : montrer tous les en-têtes du message ;

Raccourcis du menu résumé

  • Alt-S : ouvre le menu Résumé

  • Alt-X : exécute (commandes en instance, c'est seulement utile lorsque les commandes ne sont pas exécutées immédiatement) ;
  • Alt-U : mise à jour des la vue ;
  • Alt-G : aller dans les autres répertoires ;
  • Ctrl-T : vue en file ;
  • Shift-Ctrl-T : enlever la vue en file.

Raccourcis au menu OUTILS

  • Alt-T : ouvrir le menu outil ;
  • Alt-A : ouvrir le carnet d'adresse ;
  • Alt-L : ouvrir un journal de connexion.

17.2 Format de donnée du carnet d'adresse

Le carnet d'adresse est situé dans  /.sylpheed. Il est nommé addressbook.xml. Le format général du carnet d'adresse est : 

< addressbook ><p>
 < common_address ><p>
    < group name="Common addresses" ><p>
    < item ><p>
     < name > Paul < /name ><p>
     < address > paul@nlpagan.net < /address ><p>
     < remarks > Writes Sylph. manual < /remarks ><p>
        < /item ><p>
    < /group ><p>
 < /common_address ><p>
< /addressbook >
