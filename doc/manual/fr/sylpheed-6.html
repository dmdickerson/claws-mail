6. À la découverte de Sylpheed

6.1 Les options du menu de Sylpheed

Le menu de Sylpheed est le point d'entrée à toutes les opérations que vous pouvez réaliser avec Sylpheed. La structure du menu est la suivante :

Menu Fichier

  • Ajouter une boîte aux lettres... : cette option vous permet d'ajouter un nouveau répertoire de courrier MH. Il générera automatiquement dans la nouvelle boîte aux lettres les dossiers des courriers reçus (Réception), envoyés (Envoyés), en attente d'envoi (File d'attente), brouillons (Brouillon) et des courriers supprimés (Corbeille).
  • Mise à jour de l'arbre des répertoires : il relit l'arbre complet des boîtes aux lettres existantes pour qu'elles reflètent l'état correct (ce n'est pas nécessaire habituellement).
  • Dossiers : vous pouvez y ajouter, renommer ou effacer des dossiers de courrier.
  • Importer un fichier mbox... : vous pouvez importer des fichiers mbox (comme ceux créés par Pine) dans une boîte aux lettres MH.
  • Exporter vers un fichier mbox... : vous pouvez exporter une boîte aux lettres MH vers un fichier mbox.
  • Vider la corbeille : évidemment, ceci videra le dossier de la poubelle.
  • Sauvergarder comme... : vous pouvez sauvegarder les messages sélectionnés dans un fichier texte pur sur le disque dur.
  • Imprimer... : vous pouvez imprimer les messages sélectionnés vers une imprimante.
  • Fermer : arrête Sylpheed.
  • Quitter : arrête aussi Sylpheed.

Menu Édition

  • Copier : vous pouvez copier le texte sélectionné vers un tampon, à partir duquel vous pouvez le coller dans une zone différente. Raccourci : Ctrl-C
  • Tout sélectionner : sélectionne tout le texte dans la fenêtre courante. Raccourci : Ctrl-A
  • Chercher dans le message... : cherche la fenêtre courante un texte que vous choisissez. Raccourci : Ctrl-S

Menu Vue

Le menu Vue vous permet de changer la manière dont Sylpheed apparaît. Vous pouvez sélectionner un jeu différent de fenêtres séparément (vue de dossiers, messages, etc.), valider ou désactiver la barre de d'état et ainsi de suite.

  • Arbre des répertoires : pas encore implémenté
  • Vue des message : pas encore implémenté
  • Barre d'outils : configure L'apparence de la barre d'outils :avec icône et texte ou l'un des deux.
  • Barre d'état : la montre ou la cache.
  • Arbre de répertoire séparé : sélectionne entre avoir l'arbre des répertoires attaché à la fenêtre principale ou le laisser flotter séparément.
  • Séparer la vue du message : sélectionne entre avoir le message sélectionné dans le cadre du sommaire dans la fenêtre principale ou flottant séparément.
  • Jeux de caractères : vous pouvez sélectionner le jeu de caractère que vous voulez utiliser pour afficher les messages reçus.

Menu Message

Ici, vous pouvez trouver les options suivantes :

  • Relever le courrier : ceci permet de collecter tous les nouveaux courriers dans votre boîte aux lettres personnelle.
  • Relever le courrier de tous les comptes : Sylpheed scrutera toutes les identités de comptes définies pour les nouveaux courriers.
  • Envoyer les messages en file d'attente : envoie tous les messages que vous avez mis dans le dossier File d'attente.
  • Composer un nouveau message : vous pouvez écrire un nouveau courrier.
  • Répondre : un courrier sera créé avec l'expéditeur du message sélectionné dans le champ du destinataire. En fonction de la configuration, votre signature sera ajoutée au courrier et le texte de courrier original sera décalé.
  • Répondre à tous : comme Répondre mais les destinataires seront toutes les adresses du courrier original (incluant ceux dans la partie cc:).
  • Transférer : un message sera créé avec le message original dans le corps du texte. Vous pouvez alors entrer une adresse de destinataire, éditer le texte si vous le souhaitez et l'envoyer.
  • Transférer comme attachement : comme Transférer mais le texte du message sera envoyé attaché comme un fichier texte séparé.
  • Déplacer : vous pourrez déplacer le message vers un répertoire différent. Vous pouvez sélectionner le répertoire de destination depuis la fenêtre de navigation.
  • Copier : comme pour Déplacer, seul le message est copié dans le nouveau répertoire, au lieu d'être déplacé.
  • Supprimer : vous pouvez effacer le message qui est sélectionné dans le tableau du sommaire.
  • Marquer : Vous obtenez un sous-menu : Marquer, Démarquer, Marquer comme lu, Démarquez comme lu. Lorsque vous marquez les messages, vous pouvez faire des actions spécifiques (i.e. déplacer, effacer) sur tous les messages marqués. Vous pouvez démarquer les messages que vous avez marqués par erreur. Les autres options font ce qu'indiquent leur nom, elles marquent ou démarquent un message comme lu.
  • Ouvrir dans une nouvelle fenêtre : Ceci ouvrira le message du tableau du sommaire dans une nouvelle fenêtre, vous permettant de voir plus de texte en une fois. Vous pouvez fermer cette fenêtre en cliquant le bouton de fermeture ou en pressant sur la touche d'échappement.
  • Voir le source du message : ouvrir une nouvelle fenêtre avec le message dedans, incluant tous les en-têtes de message. Vous pouvez fermer cette fenêtre en cliquant sur le bouton de fermeture ou en pressant la touche d'échappement.
  • Afficher tous les en-têtes : cela ouvre une nouvelle fenêtre avec seulement les en-têtes du message. Vous pouvez fermer cette fenêtre en cliquant sur le bouton de fermeture ou en pressant la touche d'échappement.
  • Rééditer : cette option est seulement disponible lorsque vous sélectionnez un message dans le répertoire Brouillon. Vous pouvez alors éditer le message brouillon.
Note sur les fonctions comme Déplacer, Copier et Effacer : si vous ne spécifiez pas d'exécuter les commandes immédiatement (dans la configuration), vous aurez à sélectionner la commande Exécuter. C'est un bouton dans la barre d'outils et la commande peut être aussi trouvée dans le menu Sommaire.

Menu Sommaire

Dans le menu Sommaire, vous pouvez effectuer tous les types de fonctions qui sont liées au traitement des messages. Les actions sont effectuées sur les messages dans la fenêtre du sommaire des messages.

  • Supprimer les messages en double : Ceci enlèvera les messages du répertoire actif qui vous ont été envoyés deux fois. Par exemple, vous recevez un message directement et par le biais d'une liste de diffusion en même temps.
  • Filtrer les messages : vous pouvez utiliser cette option si vous souhaitez filtrer les messages existants vers un nouveau filtre que vous avez configuré.
  • Exécuter : cette option est nécessaire si vous n'avez pas configuré exécuter immediatement dans les préférences. Exécuter effectuera les actions que vous souhaitez (e.g. copier, déplacer, filtrer, etc.)
  • Mettre à jour : rafraîchit la vue des fenêtres de messages pour refléter tous changements.
  • Message précédent : sélectionne le message précédent. Les anciens utilisateurs de Pine peuvent aussi utiliser la touche « p ».
  • Message suivant : sélectionne le message suivant. Les anciens utilisateurs de Pine peuvent aussi utiliser « n ».
  • Message non lu suivant : sélectionne le message non lu suivant dans le dossier.
  • Autre dossier... : vous pouvez ouvrir un nouveau dossier de courrier. Vous pouvez utiliser une fenêtre de navigation pour ce faire.
  • Trier : cette option vous permettra de trier les vues de courriers de différentes manières. Vous pouvez trier par date, par sujet, par numéros (on parle ici des numéros internes des messages MH), etc. Notez que Attirer par sujet est différent de trier par sujet. Attirer par sujet écrase le « Re : » références dans le sujet. De cette manière, les messages sont triés d'une manière plus logique.
  • Vue par fil de conversation : les messages sont liés ensemble dans un arbre de vue, sur des sujets.
  • Vue sans fil de discussion : défait l'objet précédent.
  • Choix des éléments affichés... : montre une boîte de dialogue dans laquelle vous pouvez activer et désactiver les objets que vous voulez voir dans la fenêtre du sommaire. Dans la version 0.6.4, vous pouvez aussi déterminer l'ordre dans lequel les colonnes de la fenêtre du sommaire doivent être affichées.

Menu Outils

Dans le menu Outils, vous allez trouver deux options (jusqu'à la version 0.6.4). C'est le carnet d'adresses et la fenêtre de log.

Dans le carnet d'adresses, comme vous pouvez le souhaiter, vous pouvez stocker les adresses email que vous souhaitez garder. Vous pouvez créer des répertoires dans le carnet, pour garder des adresses ensemble.

Dans la fenêtre de log (journal de bord), vous pouvez voir ce que Sylpheed fait. Garder une trace des choses de cette manière est important lorsque vous suspectez des anomalies. Vous pouvez coller les informations de la fenêtre de log dans un courrier et l'envoyer aux développeurs dans la liste de diffusion.

Menu de Configuration

Dans le menu Configuration, clairement, vous pouvez configurer Sylpheed de la manière dont vous voulez qu'il travaille et réagisse.

  • Préférences générales... : dans cette option, il vous sera présenté un dialogue dans lequel vous pouvez sélectionner un certain nombre d'onglets. Chaque onglet représente un jeu spécifique de configurations dans Sylpheed. Dans le chapitre suivant, ces options de dialogue sont expliquées avec plus de détails.
  • Options de filtrage... : cette option vous montre un dialogue dans lequel vous pouvez configurer les règles de filtrage que Sylpheed appliquera à la réception des courriers. Vous spécifiez d'abord quelle(s) partie(s) de votre courrier Sylpheed doit contrôler et ensuite vous indiquez à Sylpheed quoi faire. Vous pouvez déplacer les messages vers un répertoire différent (incluant la corbeille pour se débarrasser des spams). Le dossier de destination est sélectionnable dans une fenêtre de navigation qui apparaît lorsque vous pressez le bouton Sélection...

    Vous pouvez aussi choisir de ne pas récupérer les courriers. Notez que cela laissera le message sur le serveur de courrier de votre prestataire, le message n'y sera pas détruit !

    Lorsque vous pressez le bouton Enregistrer, la règle de filtrage est ajoutée à la liste des règles. Si vous sélectionnez Remplacer, alors la règle sélectionnée est remplacée par l'information des configurations entrées. Enfin, le bouton Supprimer permet de détruire les règles de filtrage sélectionnées.

    Veuillez noter que vous pouvez configurer deux conditions pour une même règle. Elles peuvent être combinées par l'une des opérations logiques ou ou et. L'opération ou signifie qu'au moins une des deux conditions doit être satisfaite pour que la règle soit exécutée. L'opération et signifie que les deux conditions doivent être satisfaites simultanément pour que l'action soit effectuée.

    À droite de la liste des règles se trouvent les boutons appelés Plus haut et Plus bas. Grâce à eux, vous pouvez déplacer dans la liste les règles vers le haut ou vers le bas pour les réordonner. Ceci affectera l'ordre d'exécution des règles et a en générale une incidence sur le résultat final du filtrage, veuillez donc prendre en compte l'ordre dans lequel vous arrangez les règles.

  • Configuration du compte actuel... : ici, vous pouvez configurer les options du compte que vous utilisez actuellement, par exemple pour décider si le courrier doit être effacé après traitement, ou de quel serveur il provient, etc. Cette fenêtre de configuration est la même que celle utilisée lors de la création d'un nouveau compte.
  • Création d'un nouveau compte... : ici vous donnez un nom au nouveau compte à créer (chaque compte possède un nom reconnaissable), vous configurez votre nom et une adresse de courrier, etc. Vous configurez le type de serveur pour lequel vous avez un accès (POP3, IMAP4, boîte de courrier local, serveur de nouvelles) et où doivent aller les messages téléchargés (le répertoire par défaut est inbox). Dans l'onglet Réception vous pouvez activer ou désactiver certaines options, dans l'onglet Envoi, vous pouvez configurer la manière d'envoyer vos messages. Dans l'onglet Composition, vous pouvez désigner un fichier de signature spécifique et dans Avancée, vous pouvez configurer explicitement les ports pour les serveurs utilisés. (Ne vous compliquez la vie pas avec ceux-ci, à moins que vous ne soyez sûr d'être obligés de le faire !!)
  • Édition des comptes... : ici, vous pouvez sélectionner à partir d'une liste un des comptes que vous avez créés pour en modifier la configuration. Notez que si vous êtes en train de composer un message vous ne pouvez pas éditer le compte que vous utilisez pour ce message. Cela permet d'assurer de l'intégrité des données.
  • Choisir un autre compte : ici, vous pouvez sélectionner quel compte vous voulez utiliser pour écrire des messages.

Menu d'Aide

Ce menu vous permet d'accéder au manuel (japonais jusqu'à présent) et vous pouvez aussi voir des informations sur l'auteur et la version de Sylpheed dans le menu À propos.

6.2 Menu contextuel

La fenêtre des dossiers ainsi que celle du sommaire disposent chacune d'un menu contextuel que vous pouvez dérouler en cliquant sur le bouton 2 de la souris. (Utilisateurs de windows : le bouton 2 est le bouton droit de la souris.)

Dans la vue des dossiers, vous pouvez ajouter, renommer et enlever des dossiers par le menu contextuel.

Dans le sommaire des messages, vous pouvez faire beaucoup de choses à partir du menu contextuel, comme voir les en-têtes du message, ouvrir un message dans une fenêtre séparée (c'est aussi possible en double cliquant sur un message), etc.
