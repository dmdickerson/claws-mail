19.2 Liste de diffusion

Si vous voulez garder Sylpheed à jour et que vous voulez parler du programme avec d'autres utilisateurs, vous pouvez rejoindre la liste de diffusion.

Send a mail to sylpheed-ctl@good-day.net that says

subscribe Votre nom

dans le corps.

Le serveur retourne le courrier une fois pour le premier enregistrement de la requête qui demande la confirmation : « Puis-je vous mettre sur la liste de diffusion ? » qui inclut la phrase suivante (le nombre est un exemple).

confirm 84682771 Anne Shirley

C'est une précaution pour éviter de vous inscrire contre votre volonté.

Si vous recevez un courrier de confirmation d'entrée, envoyez un courrier qui inclut la phrase :

confirm Password(a number) Votre nom

de l'adresse pour l'enregistrement : sylpheed-ctl@good-day.net encore. Puis, il est considéré que vous avez confirmé l'enregistrement et vous êtes inscrit dans le serveur.

L'adresse pour poster dans la liste est sylpheed@good-day.net.

Notez : si vous perdez le courrier qui dit

confirm Password(a number) Votre nom

ou que vous êtes perdus et vous voulez, en d'autres mots, redémarrer depuis le début, envoyez à nouveau

subscribe Anne Shirley

à sylpheed-ctl@good-day.net.

Comment déssouscrire.

Envoyez un courrier sur sylpheed-ctl@good-day.net qui indique

unsubscribe

dans le corps.

19.4 Comment indiquez à un autre programme d'utiliser Sylpheed comme programme de gestion de courriers ?

Si vous souhaitez utiliser Sylpheed depuis Netscape ou Opera, par exemple alors, vous avez besoin d'aller dans la section « préférences » puis choisir « applications » et dans la partie client de courrier, vous aurez besoin d'entrer sylpheed --compose.
